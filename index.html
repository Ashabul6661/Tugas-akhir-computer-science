Introduction

Ini adalah tampilan awal web sederhana menggunakan HTML dengan template bootstrap. Web ini dibuat oleh Ashabul Kahfi dengan tujuan menuntaskan tugas templating web mata kuliah computer science.

Web ini berisi deskripsi singkat tentang diri saya, project atau tugas-tugas berbasis web/data yang saya kerjakan selama kuliah, dan kontak.

Tentang Saya

Halo! Saya Ashabul Kahfi, mahasiswa jurusan Bisnis Digital di Universitas Negeri Makassar. Saya memiliki ketertarikan pada dunia teknologi digital, khususnya dalam hal pengembangan web, analisis data, dan penerapan teknologi dalam kegiatan bisnis. Saat ini saya sedang aktif belajar berbagai tools dan bahasa pemrograman seperti HTML, CSS, Python, serta framework Flask dan Streamlit untuk mendukung kemampuan saya di bidang tersebut.

Saya percaya bahwa perkembangan teknologi membuka banyak peluang untuk menciptakan solusi yang efektif dan efisien, baik untuk bisnis maupun kehidupan sehari-hari. Melalui website ini, saya ingin membagikan sebagian dari proses belajar saya, termasuk proyek-proyek sederhana yang saya kerjakan sebagai bentuk pengembangan diri selama masa kuliah.
